# SQL Data Analysis: E-commerce Case Study

## About the Project
* Analysis of a modeling store database (MySQL) focused on key business metrics.

## SQL Techniques Used
* **Complex JOINS**: Combining data across customers, orders, and payments.
* **CTE (Common Table Expressions)**: Creating readable, modular performance reports.
* **Window Functions**: Building sales rankings within specific categories.
* **CASE WHEN Logic**: Automatic segmentation of products and customers.
* **Aggregations**: Advanced statistical and financial calculations.

## What's Included in the Analysis?
* Top 10 most valuable customers report.
* Product categorization based on profit margins.
* Employee performance comparison against company averages (KPIs).
* Ranking of sales leaders within each product line.

## Tools
* **Database Engine**: MySQL 8.0
* **Management**: phpMyAdmin / MySQL Workbench
