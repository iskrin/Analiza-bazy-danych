# SQL Data Analysis: E-commerce Case Study

## 📌 O projekcie

◦ Analiza bazy danych sklepu modelarskiego (MySQL) skupiona na kluczowych metrykach biznesowych. Projekt pokazuje umiejętność przekształcania surowych danych w gotowe raporty dla biznesu.

🚀 Wykorzystane techniki SQL

* Złożone złączenia (JOINS): łączenie danych o klientach, zamówieniach i płatnościach.

* CTE (Common Table Expressions): tworzenie czytelnych, modułowych raportów wydajności.

* Funkcje okna (Window Functions): budowanie rankingów sprzedaży wewnątrz kategorii.

* Logika CASE WHEN: automatyczna segmentacja produktów i klientów.

* Agregacje: zaawansowane obliczenia statystyczne i finansowe.




## 📊 Co zawiera analiza?

* Raport TOP 10 najbardziej wartościowych klientów.

* Kategoryzację produktów według marży zysku.

* Porównanie wyników pracowników ze średnią firmy (KPI).

* Ranking liderów sprzedaży w każdej linii produktowej.



## 🛠️ Narzędzia

* Silnik bazy danych: MySQL 8.0

* Zarządzanie: phpMyAdmin / MySQL Workbench
